# SNS 이모티콘 메이커 기획안

## 1. 서비스 한 줄 정의

사용자가 자신의 이미지를 업로드하고, 원하는 스타일을 선택하면 카톡·텔레그램·왓츠앱·디스코드·LINE·인스타그램에서 활용 가능한 스티커/이모티콘 이미지를 생성하는 무료 웹앱.

## 2. 핵심 방향

이 서비스는 유료 AI 서버를 운영하는 SaaS가 아니라, 사용자가 자신의 Gemini API Key를 직접 넣어 사용하는 무료 제작 도구로 시작한다.

```text
이미지 업로드
→ 스타일 선택
→ 대표 이모티콘 생성
→ 세트 생성
→ 플랫폼별 내보내기
```

## 3. 무료 배포 정책

- 기본 사용료 무료
- 회원가입 없이 사용 가능하게 설계
- Gemini API Key는 사용자 본인이 발급
- 운영자 서버에는 API Key 저장 금지
- 가능한 기능은 브라우저에서 처리
- 서버는 Gemini 호출을 중계하는 최소 역할만 수행

## 4. MVP 기능

### 1차 MVP

- Gemini API Key 입력
- 이미지 업로드
- 20개 스타일 라이브러리
- 대표 문구 1개 선택
- 선택 스타일로 대표 이모티콘 1장 생성
- PNG 다운로드

### 2차 MVP

- 8개 기본 문구 일괄 생성
- ZIP 다운로드
- 플랫폼별 사이즈 변환
- 배경 제거
- 카톡/텔레그램/왓츠앱/디스코드/LINE/인스타 내보내기 프리셋

### 3차 확장

- Chrome Extension
- 우클릭 이미지 가져오기
- SNS 웹페이지에서 바로 스티커 생성
- 스타일 프롬프트 공유 라이브러리

## 5. 대표 사용자 흐름

1. 사용자가 Gemini API Key를 입력한다.
2. 사진 또는 캐릭터 이미지를 업로드한다.
3. 스타일 카드 20개 중 하나를 선택한다.
4. 대표 문구를 선택한다.
5. AI가 대표 이모티콘 1개를 생성한다.
6. 사용자가 결과를 확인한다.
7. 마음에 들면 전체 세트를 생성한다.
8. 원하는 플랫폼 규격으로 다운로드한다.

## 6. 스타일 카테고리

- 귀여움: 치비, 한국 메신저풍, 일본 카와이, K-팬덤
- 밈: 끄적끄적 낙서, 막대인간, B급 병맛, 밈 반응짤
- 예술: 웹툰, 수채화, 종이 오려붙이기, 그림일기
- 최신/트렌드: 3D 피규어, 클레이, 픽셀 아트, 글리치, 그래피티
- 미니멀: 플랫 이모지
- 반려동물: 반려동물 전용

## 7. 확장프로그램 방향

브라우저 확장프로그램은 웹앱의 보조 진입점으로 사용한다.

예상 기능:

- 이미지 우클릭 → 이 이미지로 스티커 만들기
- 현재 탭 이미지 가져오기
- 웹앱으로 이미지 전달
- 스타일 선택 후 생성

## 8. 개발 원칙

- 무료 배포 우선
- 운영자 API 비용 최소화
- 사용자의 키는 저장하지 않음
- 플랫폼별 자동 변환 기능은 점진적으로 추가
- 처음부터 완벽한 카카오 심사 제출 자동화가 아니라, 제출 준비용 도구로 설계
